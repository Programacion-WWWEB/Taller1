Endpoints de Álbum

/Album/Lista

Crea un listado de los álbumes que están en la base de datos, retorna el id del álbum, el nombre, el artista el tipo, la fecha de estreno, el rating, el lenguaje, el genero y el color del álbum.

/Album/Buscar/{id}

Se realiza una búsqueda que entrega un álbum especifico a partir del id que se le remplaza por el “{id}”, y retorna el id del álbum, el nombre, el artista el tipo, la fecha de estreno, el rating, el lenguaje, el género y el color del álbum.

/Album/Agregar

Se le entrega un JSON con los datos del álbum, también se le puede directamente entregar un listado de los tracks que posee el álbum y los agregara a la tabla de tracks en la base de datos. No se debe agregar un id.

/Album/Actualizar

Se le entrega un JSON con la información del álbum, se requiere un id para saber que álbum se esta editando, si no se guarda la información como un álbum nuevo.

/Album/Borrar

Se le entrega un JSON con toda la información del álbum que se quiere borrar y lo elimina de la base de datos.

/Album /Borrar/{id}

Se recibe el id del álbum a borrar en la tabla. Para hallar el álbum especifico se usa el número que remplaza “{id}”.
